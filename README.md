#  GPA Calculator

Modern web tool for calculating university GPAs for Computer Science students.

##  Features
- **Modern UI** with light/dark modes
- **All phases & specializations** supported
- **Automatic calculations** (semester & annual GPA)
- **Data saved automatically** in browser
- **Mobile-friendly** responsive design

##  Quick Use
1. Select study path (Phase ‚Üí Specialty ‚Üí Semester)
2. Enter your grades
3. Get instant GPA results

##  Specializations
- Preparatory (S1-S4)
- Data Science, Software Engineering, AI
- Networks, Cybersecurity, Information Systems
